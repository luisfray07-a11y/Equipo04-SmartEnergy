\## Descripción

\## Tecnologías

\- Arduino + Sensores DHT11/TP4056

\- Python + InfluxDB + Grafana

\- Energía Solar + Baterías Li-ion

\- `docs/` - Documentación técnica

\- `hardware/` - Esquemáticos y componentes

\- `grafana/` - Dashboards y queries
